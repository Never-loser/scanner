# 🛡️ CleanIP Bot

یک ربات تلگرامی برای جمع‌آوری و توزیع آیپی‌های تمیز کلودفلر. کاربران آیپی تمیز ارسال می‌کنند، ادمین تأیید می‌کند، و ربات به صورت خودکار آیپی را داخل کانفیگ‌های Vless و Trojan جایگزین می‌کند.

## ✨ امکانات

- ارسال آیپی تمیز کلودفلر از طریق تلگرام
- سیستم تأیید/رد توسط ادمین با دکمه‌های inline
- جایگزینی خودکار آیپی داخل کانفیگ‌های Vless و Trojan
- پشتیبانی از IPv4 و IPv6
- ذخیره‌سازی با SQLite
- آمار مشارکت کاربران

## 🚀 راه‌اندازی

### پیش‌نیازها

- Python 3.9 یا بالاتر
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather))
- آیدی عددی تلگرام (به عنوان ادمین)

### نصب

```bash
git clone https://github.com/Never-loser/scanner.git
cd scanner
pip install -r requirements.txt
```

### تنظیمات

یک فایل `.env` در پوشه پروژه بسازید:

```env
BOT_TOKEN=توکن_ربات_تلگرام
ADMIN_ID=آیدی_عددی_تلگرام
```

### اجرا

```bash
python bot.py
```

## 📁 ساختار پروژه

```
├── bot.py           # منطق ربات و هندلرهای تلگرام
├── database.py      # عملیات دیتابیس SQLite
├── ip_parser.py     # پارسر کانفیگ و جایگزین‌کننده آیپی
├── seed_ips.py      # اسکریپت برای بارگذاری اولیه آیپی‌ها
├── requirements.txt
└── .env             # متغیرهای محیطی
```

## 🤖 دستورات ربات

| عملکرد | توضیح |
|--------|-------|
| `/start` | نمایش منوی اصلی |
| 📤 ارسال کانفیگ | کانفیگ Vless/Trojan خودت رو بفرست تا آیپی تمیز داخلش جایگزین بشه |
| 📥 ارسال آیپی تمیز | یک آیپی تمیز کلودفلر برای بررسی ادمین ارسال کن |
| 📊 آمار من | ببین چند تا آیپی تأیید شده داری |

## 🔒 نحوه کارکرد

1. کاربر یک آیپی تمیز کلودفلر ارسال می‌کند
2. ربات آن را با دکمه‌های تأیید/رد برای ادمین می‌فرستد
3. پس از تأیید، آیپی وارد دیتابیس می‌شود
4. هر کاربر می‌تواند کانفیگ پروکسی خود را بفرستد و آیپی تمیز تصادفی داخلش جایگزین شود
